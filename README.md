# bookstop-home

Homepage for BookStop.

https://bookstop.app

## Development

Install all dependencies using:

```bash
npm i
```

Then, run the development server with 2 terminals:

```bash
npm run dev
```

```bash
npm run tw:watch
```

## Deployment

Build the app:

```bash
npm run build
```

Start the server:

```bash
npm run serve
```
